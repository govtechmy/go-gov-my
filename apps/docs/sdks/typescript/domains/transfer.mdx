## Arguments

<ParamField body="slug" type="string">
  Name of the domain to transfer. Eg. `go.gov.my`.
</ParamField>

<ParamField body="newWorkspaceId" type="string">
  The ID of the new workspace to transfer the domain to.
</ParamField>
