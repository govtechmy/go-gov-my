---
title: "Delete"
"og:title": "go.gov.my.domains.delete - Delete a domain - TypeScript SDK"
description: "Delete a domain from a workspace. It cannot be undone. This will also delete all the links associated with the domain."
---

## Arguments

<ParamField body="slug" type="string" required>
  Name of the domain. Eg. `go.gov.my`.
</ParamField>

## Response

<ResponseField name="slug" type="string" required>
  Name of the domain that was deleted.
</ResponseField>

<RequestExample>

```ts
await GoGovMy.domains.delete("go.gov.my");
```

</RequestExample>

<ResponseExample>

```ts
{
  slug: "go.gov.my";
}
```

</ResponseExample>
