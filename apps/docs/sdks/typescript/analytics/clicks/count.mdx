## Response

The number of clicks matching the specified queries.

<RequestExample>

```typescript
await go.gov.my.analytics.clicks.count({
  key: "dummy",
  domain: "go.gov.my",
  interval: "30d",
});
```

</RequestExample>

<ResponseExample>

```typescript
20;
```

</ResponseExample>
