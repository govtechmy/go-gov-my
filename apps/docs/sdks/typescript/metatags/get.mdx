## Arguments

<ParamField body="url" type="string" required>
  The URL to retrieve metatags for.
</ParamField>

## Response

<ResponseField name="title" type="string | null" required>
  The meta title tag for the URL.
</ResponseField>

<ResponseField name="id" type="string | null" required>
  The meta description tag for the URL.
</ResponseField>

<ResponseField name="id" type="string | null" required>
  The OpenGraph image for the URL.
</ResponseField>

<RequestExample>

```ts
await go.gov.my.metatags.get({
  url: "https://go.gov.my",
});
```

</RequestExample>

<ResponseExample>

```ts
{
  title: 'GoGovMy - Link Management for Modern Marketing Teams',
  description: 'GoGovMy is the open-source link management infrastructure ...',
  image: 'https://assets.go.gov.my/thumbnail.jpg'
}
```

</ResponseExample>
