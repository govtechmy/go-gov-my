---

import AnalyticsArguments from "/snippets/analytics-arguments.mdx";

<Note>
  Analytics endpoints require a [Pro plan](https://d.to/pro) subscription or
  higher.
</Note>

<AnalyticsArguments />

## Response

<ResponseField name="url" type="string" required>
  The destination URL
</ResponseField>

<ResponseField name="clicks" type="number" required>
  The number of clicks from this URL
</ResponseField>

<RequestExample>

```typescript
await go.gov.my.analytics.clicks.topUrls({
  key: "dummy",
  domain: "go.gov.my",
  interval: "30d",
});
```

</RequestExample>

<ResponseExample>

```typescript
[
  { url: "https://go.gov.my/", clicks: 13 },
  { url: "https://go.gov.my/?qr=", clicks: 4 },
];
```

</ResponseExample>
